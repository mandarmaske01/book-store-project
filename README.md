# BookStore Management System

Welcome to the BookStore Management System project! This system is designed to efficiently manage books, showcasing a comprehensive full-stack web development skill set.

## Features

-   Server-Side Logic:   Implemented using Servlets for seamless functionality.
-   Dynamic Web Pages:   Utilized JSP for dynamic web page generation.
-   Engaging UI:   Designed an interactive user interface with HTML/CSS.
-   Backend Processing:   Leveraged Java for efficient backend processing.
-   Database Management:   MySQL ensures a reliable and effective database.

## Project Structure

-   /src:   Contains the source code for Servlets and Java backend.
-   /web:   Includes HTML/CSS files for the user interface.
-   /database:   Holds MySQL scripts for database setup.


## Dependencies

- Java
- Servlet Container (e.g., Apache Tomcat)
- MySQL
